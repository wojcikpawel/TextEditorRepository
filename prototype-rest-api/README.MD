## Run REST-API:
* Run Node.js Server:
```sh
node server.js
```
* Run Browser or Postman and type following endpoints:
```sh
http://localhost:5000/EndpointsOrders/komunikatzamowienie
```
```sh
http://localhost:5000/EndpointsProducts/komunikatprodukt
```
